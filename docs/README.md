# distill documentation

[`@interactkit/distill`](https://github.com/InteractKit/distill) - an LLM writes
an entity-extractor program for you once, then you run that program (no LLM) to
pull structured data out of text.

New here? Read **[How it works](how-it-works.md)** first - it walks through a real
example end to end.

### Guides

- **[How it works](how-it-works.md)** - label → build → run, explained with an example.
- **[CLI reference](cli.md)** - every command and flag (`init`, `build`, `label`, `run`, `host`).
- **[Configuration](configuration.md)** - the config file, field by field.
- **[Input & output](input-output.md)** - data formats in and JSON out.
- **[Providers](providers.md)** - OpenAI, Anthropic, Google, and local Ollama.
- **[Hosting](hosting.md)** - serve over HTTP; learn from live traffic; metrics.

Or jump to the [Quick start](https://github.com/InteractKit/distill#quick-start).
